тЬЕ Key Insight
Arrow functions: capture this from where they are defined.
Normal functions: this depends on how they are called.
Strict mode: prevents automatic substitution with global object.
DOM handlers: this is the element, unless you use an arrow function.

тЬЕ This gives you a quick reference:
In global scope, this is the global object (window in browsers).
In strict mode, standalone functions have this = undefined.
In object methods, this points to the object.
With call/apply/bind, you explicitly set this.
Arrow functions inherit this from their surrounding scope.
In DOM event handlers, this is the element (unless you use an arrow function, which captures outer scope instead).
